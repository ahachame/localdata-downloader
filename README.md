# LOCALDATA 음식점 인허가 데이터 다운로더

지방행정인허가데이터개방(LOCALDATA) 포털이 공공데이터포털(data.go.kr)로 이관됨에 따라 발생하는 다운로드 연결 에러를 해결하고, 대용량 데이터를 날짜별로 실시간 필터링하여 다운로드할 수 있도록 설계된 GUI 기반 윈도우 유틸리티 프로그램입니다.

## 주요 기능
- **공공데이터포털 보안 우회:** 차단 정책(403 Forbidden)을 회피하기 위한 Referer 및 User-Agent 헤더 처리 탑재.
- **실시간 대용량 파일 스트리밍 필터링:** 650MB가 넘는 원본 데이터를 하드디스크에 통째로 저장하지 않고, 실시간으로 한 행씩 읽어 들여 설정한 인허가일자 범위에 부합하는 레코드만 걸러내어 저장 (메모리 및 디스크 용량 최소화).
- **사용하기 편리한 GUI:** 저장 경로 지정, 다운로드 대상 선택(일반음식점/휴게음식점), 기간 설정(최근 30일 기본 제공), 프로그레스 바 및 실시간 로그 콘솔 제공.
- **독립 실행형 파일:** 파이썬 환경이 없는 일반 사용자도 바로 실행 가능한 `.exe` 파일 제공.

## 실행 방법
1. Releases 메뉴에서 최신 `4.6_localdata_downloader.exe` 파일을 다운로드합니다.
2. 실행 시 윈도우 SmartScreen 보안 경고창이 뜰 경우 '추가 정보' -> '실행'을 클릭합니다.
3. 원하는 데이터 종류와 날짜 범위를 선택한 뒤 '다운로드 시작'을 클릭합니다.
